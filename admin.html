How this page works
  • Four sections below: Source Overview (live stats) → Source Requests queue → Keyword Changes queue → Source Fixes (broken sources).
  • Push a source request to auto-apply it (works for RSS feeds & subreddits). Complex requests (new APIs) need Olli — the button is disabled on those.
  • Apply a keyword request to save it to keywords.txt, or use the Quick add form to add a keyword directly. Decline dismisses without applying.
  • Scrapes run automatically at 08:00 and 18:00 UTC. Use Run scrape now for an immediate refresh (~2–3 min).
  • Admins and DAO members alike submit new sources & keyword weight-changes at /suggest (no login required). They land in the queues here.

Custom domain setup — how to point qubie.quantumbiologydao.xyz at this site

This needs to happen once, by whoever manages DNS for quantumbiologydao.xyz.

Step 1 — Olli adds the custom domain on Railway

Have Olli add qubie.quantumbiologydao.xyz as the custom domain on Railway. Railway will return a CNAME target (looks like xyz.up.railway.app). Copy it.

Step 2 — Hand off the DNS record to the designer / dev team

Copy and paste this record into wherever DNS for quantumbiologydao.xyz is managed:

FieldValue
TypeCNAME
Name / Hostqubie
Value / Targetthe CNAME target from Step 1
TTLdefault / auto

If the DNS is on Cloudflare: set the record to “DNS only” (gray cloud), not “Proxied” — otherwise Railway can't provision SSL.
