How to contribute
  • QUBIE News is built by the DAO. Anyone can suggest new sources (journals, preprints, YouTube channels, forums) or new keywords — no login required.
  • Sources are where we look. Keywords are what we consider relevant. Each keyword has a weight 1–10: higher = article mentioning it ranks higher in the feed. Currently quantum biology and radical pair are core terms worth weighting heavily.
  • Check before suggesting. Sources we already pull from are in the left column below; keywords already in the scoring system are in the right cloud. If it's already there, no need to re-add.
  • Click any existing keyword tag to request a weight change — e.g. if you think cryptochrome should count for more, click it, pick a new weight, submit with your reasoning.
  • Suggest anything missing via the two forms below. Olli reviews everything in the admin queue. Approved changes take effect on the next scrape (within 12 hours, or sooner if Olli triggers a manual one).
